i5/OS Programmer's Toolkit: Queue Tools

0.2.16

i5/OS Programmer's Toolkit is a set of tools designed to easy programmers' daily works on an IBM i box(previous known as AS/400, iSeries).

Go back to main page of i5/OS Programmer's Toolkit.

Members of i5toolkit

Tools currently avaialble are listed as the following:
  • Queue Tools
    • CL Command DSPQD (Display Queue Description)
    • CL Command DSPQMSG (Display Queue Messages)
    • CL Command CLRUSRQ (Clear User Queue)
    • CL Command ENQUSRQ (Enqueue User Queue)
    • The DEQUSRQ (Dequeue User Queue) API

Get Help Information

All commands and displays provided by this project supports online help. One can abtain help information by pressing F1 in each command prompt display or UIM panel.
