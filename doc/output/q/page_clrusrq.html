CL Command CLRUSRQ (Clear User Queue)

Go back to main page of i5/OS Programmer's Toolkit.

The CL command CLRUSRQ clears all messages on a user queue object.

For example, the following command clears user queue Q61 

CLRUSRQ Q(Q61)
